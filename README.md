# Python-Fundamental-1
Python Fundamental 1 (extra Setembro) - Turma 2025B

Plataforma Apreda Mas - https://aprendamais.mec.gov.br/course/view.php?id=2646
Esse curso aborda assuntos como introdução à programação e à linguagem Python, dados e operadores, variáveis e interação com o usuário e estruturas condicionais. O curso foi desenvolvido pelos professores Vinícius Dal Bem e Ramão Tiago Tiburski, ambos do Instituto Federal de Santa Catarina (IFSC).
